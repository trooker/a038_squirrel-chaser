The com.extension.g38_otg_serial.aix extension was generated using the g038_otg_serial.java
program.  The java program employs the "usbserial.jar" library for communication over
the OTG-USB to the Orion (the Makeblock version of the Arduino UNO board).

The *.aix must be imported into the MIT Application2Innovator program project environment
to be produce Android apps.
